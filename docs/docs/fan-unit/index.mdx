## Fan Units

<Tabs groupId="fanunit" defaultValue="smart">
    <TabItem value="smart" label="Smart">
        <h2>Smart Fan Unit</h2>
        The Smart Fan Unit enhances your Compute Blade setup by providing numerous functions, features, and the flexibility to customize the operating logic.

        It incorporates an additional Raspberry Pi [RP2040](https://www.raspberrypi.com/products/rp2040/), to expand the capabilities of your system.
    </TabItem>
    <TabItem value="sta" label="Standard">
        <h2>STA Fan Unit(Standard)</h2>
        Simplified variant of the smart fan unit, featuring the same form factor and fan, but without the capability to execute or be controlled by custom applications.
        
        The STA fan unit is powered and controlled directly by the compute blade, without the need for any additional wiring.

    </TabItem>
</Tabs>

## Features
<Tabs groupId="fanunit" defaultValue="smart">
    <TabItem value="smart" label="Smart">
        - UART Compute Blade Interface
        - Fan Unit Connector
        - PWM Fan Header
        - Raspberry Pi RP2040
        - Reset Button
        - BOOTSEL Button
        - Programmable Button
        - Status LEDs
        - Temperature Sensors

        :::info
        When using the Smart Fan Unit PWM Can not be used to control the fan speed
        :::

    </TabItem>
    <TabItem value="sta" label="Standard">
        - PWM Compute Blade Interface
        - Fan Unit Connector
        - PWM Fan Header

        :::info
        When using the standard Fan Unit only the [port A(left)](/fan-unit/GPIO#blade-connector) can be used for PWM
        :::

    </TabItem>
</Tabs>

## The Fan

{/* Add Image of Fan */}

Fan Units ship with a Noctua [NF-A4x20 5V](https://noctua.at/en/nf-a4x20-5v-pwm) a 5 Volt PWM fan. with a Max power draw of 0.1 Amp

### Specification
|  Specification                           |                                          |
| :--------------------------------------- | :--------------------------------------- |
| Size                                     | 40x40x20 mm                              |
| Mounting hole spacing                    | 32x32 mm                                 |
| Connector & pin-configuration            | 4-pin PWM                                |
| Bearing                                  | SSO2                                     |
| Blade geometry                           | A-Series with Flow Acceleration Channels |
| Frame technology                         | AAO (Advanced Acoustic Optimization)     |
| Rotational speed (+/- 10%)               | 5000 RPM                                 |
| Min. rotational speed @ 20% PWM (+/-20%) | 1100 RPM                                 |

| Input power (max.)                       | 0.5 W                                    |
| Input current (typical)                  | 0.08 A                                   |
