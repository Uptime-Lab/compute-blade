# General-Purpose Input/Output
The Compute Blade allows for the uses of 11 of the Compute Module's GPIO pins.
The GPIO allows for low speed modules and additional device control.

### GPIO Locations and Functions
::::note
Functions are configurable and do not have to match the given functions.
    :::warning
    Using internal GPIO for unintended uses may lead to damage.
::::

|   GPIO                                        | Location                                                  |  Functions     |
| :-------------------------------------------: | :-------------------------------------------------------- | :------------- |
| [2](https://pinout.xyz/pinout/pin3_gpio2/)    | Expansion Module Port                                     | I2C1 SDA       |
| [3](https://pinout.xyz/pinout/pin5_gpio3/)    | Expansion Module Port                                     | I2C1 SLC       |
| [4](https://pinout.xyz/pinout/pin7_gpio4/)    | Expansion Module Port                                     | GPCLK0         |
| [12](https://pinout.xyz/pinout/pin32_gpio12/) | Fan Unit Connector                                        | PWM0, UART5 TX |
| [13](https://pinout.xyz/pinout/pin32_gpio12/) | Fan Unit Connector                                        | PWM1, UART5 RX |
| [14](https://pinout.xyz/pinout/pin32_gpio12/) | Expansion Module Port, UART Front(3pin), UART Back(4pin)  | UART0 TX       |
| [15](https://pinout.xyz/pinout/pin32_gpio12/) | Expansion Module Port, UART Front(3pin), UART Back(4pin)  | UART0 RX       |
| [18](https://pinout.xyz/pinout/pin12_gpio18/) | Internal                                                  | LEDs Control   |
| [20](https://pinout.xyz/pinout/pin38_gpio20/) | Internal                                                  | Edge Button    |
| [21](https://pinout.xyz/pinout/pin40_gpio21/) | Internal                                                  | LED Control    |
| [23](https://pinout.xyz/pinout/pin16_gpio23/) | Internal                                                  | PoE+ Indicator |

## Pinout Guide
:::tip
View an interactive guide to the full Raspberry Pi GPIO pinout on [Pinout.xyz](https://pinout.xyz/)
:::
<Tabs groupId='pinout'>
    <TabItem value="Expansion" label="Expansion Module Port" default>
        The Expansion Module Port is pin 1 through pin 10 of the Raspberry Pi GPIO pinout.
        This makes the Expansion Module Port compatible with many existing products like Real Time Clock Modules or I2C Devices.
        ![Expansion Module Port](/img/hardware/expansion-module-port.svg)
    </TabItem>
    <TabItem value="uart" label="UART" default>
        There are two dedicated sets of pins for UART0. The V5 line on UART Back can operate as an Input or Output.

        <div className="center-i">
            <img src={useBaseUrl("/img/hardware/uart-front.svg")} alt="UART Front"/>
            <img src={useBaseUrl("/img/hardware/uart-back.svg")} alt="UART Back"/>
        </div>

        <Admonition type="warning" title="Warning">
            <p>When no power source is connected to the compute blade the V5 pin can only be an input.</p>
        </Admonition>
    </TabItem>
    <TabItem value="fan" label="Fan Unit Connector" default>
        The square connector at the end of the Blade.
        The main purpose is to connect to a Fan Unit. It can be used to control any 5v PWM fan or as an additional UART port (`UART5` on CM4)

        ![Fan Unit Port](/img/hardware/fan-port.webp)
    </TabItem>
</Tabs>
