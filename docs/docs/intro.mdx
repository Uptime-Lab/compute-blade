## Compute Blade Models
The Compute Blade comes in three are hardware variants.

| Feature                   |  Basic     |   TPM  |  Dev   |
| :------------------------ | :--------: | :----: | :----: |
| CM4 Port                  | ✅        | ✅     | ✅    |
| M.2 Slot M Key up to 2210 | ✅        | ✅     | ✅    |
| Gigabit Ethernet          | ✅        | ✅     | ✅    |
| PoE++ support             | ✅        | ✅     | ✅    |
| UART                      | ✅        | ✅     | ✅    |
| Programmable Button       | ✅        | ✅     | ✅    |
| Programmable LEDs         | ✅        | ✅     | ✅    |
| USB-A 2.0                 |           | ✅     | ✅    |
| TPM 2.0                   |           | ✅     | ✅    |
| HDMI                      |           |         | ✅    |
| USB-C for Flashing CM4    |           |         | ✅    |
| MicroSD Card              |           |         | ✅    |
| Wi-Fi                     | Locked | Locked | Switchable |
| Bluetooth                 | Locked | Locked | Switchable |
| Write Protection          | Locked | Locked | Switchable |
