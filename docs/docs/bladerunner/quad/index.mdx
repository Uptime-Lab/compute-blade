# Quad

![BladeRunner™️ Quad](/img/bladerunner/quad/bladerunner-quad.webp)

The Quad BladeRunner™️ is our recommend enclosure for Homelab and cluster development.
Capable of holding four Compute Blades and two Fan Units, it is ideal for quickly spinning up a development environment.
